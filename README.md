# 🏆 1xLite | Professional Sports Betting Interface

1xLite is a premium, high-performance sports betting landing page designed with a focus on speed, user experience, and modern aesthetics. It features a sleek "Midnight Navy" theme with glassmorphism elements and smooth CSS transitions.

![Preview](banner.jpg)

## 🚀 Key Features

- **Dynamic Live Feed**: Real-time sports event cards for Football, Basketball, and Tennis.
- **Glassmorphism UI**: Premium frosted glass effects and vibrant neon accents.
- **Multi-Method Registration**: Supports both "One-Click" and "By Email" registration workflows.
- **Responsive Layout**: Fully optimized for Desktop, Tablet, and Mobile devices.
- **Pure CSS Modals**: Lightweight login and registration system without heavy JavaScript dependencies.
- **Interactive Odds**: User-friendly odds selection with instant visual feedback.

## 🛠️ Technology Stack

- **Frontend**: HTML5, Vanilla CSS3 (Custom Variables, CSS Grid, Flexbox).
- **Typography**: Inter (Google Fonts) for a modern, clean look.
- **Animations**: CSS Keyframes for live indicators and smooth transitions.
- **Icons**: Custom SVG and Emoji-based iconography for lightweight performance.

## 📁 Project Structure

## 📦 Installation & Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/abidali72/1xbet-Frontend-Master.git
   ```
2. Open `1xbet_landing.html` in any modern web browser.
3. To view the professional variant, open `professional_landing.html`.
